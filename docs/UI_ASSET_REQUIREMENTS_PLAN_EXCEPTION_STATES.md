# 方案异常状态缺失资源需求

基准效果图目录：

`img/ui/彩果_晴空汽水紧凑版V2_完整补充交付包/效果图/04_方案异常状态`

当前交付包未提供以下 3 个状态插画。页面已使用代码内线性图标兜底，后续取得资源后可替换为正式插画。

## 通用输出规范

- 优先输出 SVG，同时输出透明背景 PNG（@1x、@2x、@3x）。
- SVG 使用 `viewBox="0 0 160 120"`，不要包含页面背景、卡片边框或按钮。
- 画面安全区 144×104，四周至少保留 8px 透明空间。
- 线条圆角、轻量渐变，风格与“晴空汽水”一致。
- 主色 `#5797F5`，浅蓝 `#EAF4FF`，描边 `#C8D8EA`，薄荷点缀 `#61D6BF`。
- 禁止文字、Emoji、字体图标和写死的白色大底。

## 1. `ill_plan_empty`

- 用途：方案管理没有任何保存方案。
- 构图：打开的文件夹、上方一张方案纸、右下角半张票券；周围可有少量星光与圆点。
- 状态语义：轻松、可开始创建，不应表现为错误。
- 参考：`12_方案管理_空列表.png`。

输出：

- `ill_plan_empty.svg`
- `ill_plan_empty@1x.png`（160×120）
- `ill_plan_empty@2x.png`（320×240）
- `ill_plan_empty@3x.png`（480×360）

## 2. `ill_plan_search_empty`

- 用途：已有方案，但名称、标签或筛选条件没有匹配结果。
- 构图：文件夹或资料盒叠加放大镜，底部可有很浅的椭圆投影。
- 状态语义：没有找到，不是数据丢失或系统错误。
- 参考：`13_方案管理_搜索无结果.png`。

输出规格同上，文件名前缀改为 `ill_plan_search_empty`。

## 3. `ill_plan_load_error`

- 用途：本地数据库读取失败且没有可显示的缓存方案。
- 构图：云朵轮廓内放感叹号，使用珊瑚红错误色。
- 主错误色 `#FF5B5B`，浅背景 `#FFF0F2`。
- 状态语义：读取失败，需要用户重试；不要使用断网 Wi-Fi 图形，以免和离线缓存状态混淆。
- 参考：`14_方案管理_加载失败.png`。

输出规格同上，文件名前缀改为 `ill_plan_load_error`。
