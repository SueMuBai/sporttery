## 1. 当前结论

最终视觉基准：

`img/ui/vote/彩果选票_晴空汽水_方案管理紧凑版完整交付包`

该交付包包含的 23 个 SVG 图标已经全部复制到 `src/assets/ui/plans/`，当前没有缺失图标。按钮、卡片、菜单、Toast、Sheet、输入框、Chip、描边和分割线继续由代码绘制，不应生成整块背景切图。

目前缺少的是“紧凑版 V2”风格下的账单、选票、设置及异常状态效果图。旧版晴空汽水效果图含气泡、星光、柔光和大面积渐变，与最终 Compact V2 规范冲突，不再作为最高优先级。

## 2. 通用生成规范

- 设计宽度：390dp。
- 建议效果图画布：852～864px × 1821～1846px。
- 页面背景：`#F7FAFF`。
- 卡片背景：`#FFFFFF`。
- 普通描边：`#DCE6F2`，1px。
- 分割线：`#E8EEF6`，1px。
- 主色：`#5797F5`。
- 薄荷绿：`#61D6BF`。
- 浅紫：`#9A91F5`。
- 危险红：`#FF5B67`。
- 正文：`#1D2530`。
- 辅助文字：`#6F7B8B`。
- 二级导航：44dp，标题17sp。
- 一级标题：22sp；区块标题15sp；卡片标题14sp；正文13sp；辅助11sp。
- 页面水平边距12dp；卡片padding 10dp；卡片间距8dp。
- 移除气泡、星光、柔光、装饰性渐变和大面积主题头图。
- 顶部状态栏使用白色背景、深色系统图标。
- 底部导航沿用现有“账单 / 选票 / 设置”默认态和选中态切图。

## 7. 切图要求

仅当新增了当前图标库不存在的新语义图标时生成切图。优先提供：

- SVG：24×24 viewBox，`currentColor`，stroke-width 1.8，round端点。
- PNG：透明背景，@1x/@2x/@3x。
- 文件名使用 `ic_功能名.svg`。

不要生成：

- 卡片整图。
- 按钮整图。
- 输入框整图。
- Toast整图。
- Sheet整图。
- 描边、分割线或圆角背景图。
