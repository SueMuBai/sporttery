# Android 2.0 验收清单

> 本清单只在用户明确要求“提交构建”后执行。没有 APK 和真机证据时，阶段 8/9 不得标记完成。

## 1. 构建前门禁

```bash
npm ci
npm run verify:migration
npm test
npm run lint
npm run android:sync
npm run verify:runtime
```

必须满足：

- 迁移校验通过。
- 自动测试全部通过。
- TypeScript、ESLint、Stylelint 无错误。
- Capacitor 发现 SQLite、App、Filesystem、Share 四项插件。
- Android 资源目录使用 Vite `dist`，不存在 Python、Chaquopy 或 localhost server。

## 2. GitHub Actions 构建

手动运行 `Build signed Android APK`：

- `version_name`：例如 `2.0.0`。
- `version_code`：必须大于已安装版本，例如 `2`。

保存以下证据：

- Workflow Run URL 和 Run ID。
- 所有步骤的绿色状态。
- `app-release.apk`。
- `app-release.apk.sha256`。
- APK 文件大小和 SHA-256 校验值。

## 3. 新安装验收

在没有安装旧版本的设备执行：

1. 安装签名 APK。
2. 首次启动不联网。
3. 确认账单、选票、设置三个主导航可进入。
4. 本地空数据状态正确，不出现白屏或 localhost 连接失败。
5. 恢复网络，执行一次数据更新。
6. 选择两场比赛、选择过关、保存已购方案。
7. 彻底结束 App 后重新启动，确认比赛、方案和账单仍存在。

## 4. 旧版本覆盖升级与迁移

准备一台安装旧稳定版且包含真实测试数据的设备：

1. 记录旧版 `caiguo.db` 文件大小和以下数量：标签、方案、选择项、比赛、赛果、账单。
2. 不卸载旧版，直接覆盖安装 2.0 APK。
3. 第一次启动时保持离线，避免网络同步掩盖迁移问题。
4. 确认旧方案、标签、账单和金额全部可见。
5. 对照迁移前后数量。
6. 抽查至少 3 个方案的选择项、赔率、过关、倍数和标签。
7. 抽查账单投入、回款和盈亏，金额必须精确到 0.01 元。
8. 再次结束并启动 App，确认迁移不会重复执行或产生重复数据。
9. 确认旧 `filesDir/caiguo.db` 仍保留，未被删除或覆盖。

## 5. 生命周期与系统交互

- Popup 打开时按返回键：只关闭 Popup。
- Dialog 打开时按返回键：执行取消，不确认危险操作。
- 二级设置、方案详情按返回键：返回上一级。
- 账单、选票、设置根页面按返回键：退出 App。
- 切到后台 30 秒再恢复：当前选票草稿、滚动位置和编辑内容不丢失。
- 旋转屏幕或触发配置变化：不白屏、不重复迁移。
- 键盘弹出时打开和关闭底部面板：输入框及按钮不被遮挡。

## 6. 离线、弱网和恢复

- 离线冷启动能够读取本地数据。
- 离线点击刷新显示失败提示，但本地比赛不消失。
- 同步中断后恢复网络并重试，已成功保存的数据不重复。
- 历史请求部分失败时显示部分失败数量，比赛列表仍可操作。
- 重复快速点击刷新只执行一个同步任务。

## 7. 文件导出

- JSON 导出能调起系统分享/保存面板。
- Markdown 导出能调起系统分享/保存面板。
- 保存后文件名、扩展名和内容正确。
- JSON 能被重新解析，数量与 App 页面一致。
- 取消系统分享不导致 App 崩溃或卡在 Loading。

## 8. 界面与设备尺寸

至少覆盖等效宽度：

- 360 × 800。
- 390 × 844。
- 412 × 915。

每档保存账单、选票五种玩法、设置、日期筛选、方案详情截图。检查：

- 状态栏不遮挡标题。
- 底部系统导航不遮挡投注栏或底部导航。
- 无横向滚动。
- 所有边框和图标可见。
- 按钮文字水平和垂直居中。
- 系统字体 115% 时不截断关键金额和操作按钮。

## 9. 失败回滚

出现任何阻断问题时：

1. 不发布 2.0 APK。
2. 保留失败 Run 和日志。
3. 不删除设备旧数据库。
4. 回装 `legacy-stable-1.0.1` 对应 APK。
5. 记录问题设备、Android 版本、操作步骤、截图和数据库数量差异。

## 10. 完成判定

以下条件全部满足后，才能将阶段 8、阶段 9及线程目标标记完成：

- 签名 APK 构建成功。
- 新安装通过。
- 覆盖升级迁移通过。
- 冷启动、离线、后台恢复和返回键通过。
- 系统分享导出通过。
- 三档真机截图通过。
- 无阻断级问题。
